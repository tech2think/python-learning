# Módulo 1 — Fundamentos Python

Este módulo apresenta os fundamentos da linguagem Python com exemplos práticos do setor de distribuição de energia elétrica.

## Notebooks

| Arquivo | Conteúdo | Duração |
|---------|----------|---------|
| `01_variaveis_tipos.ipynb` | Variáveis, tipos de dados, estruturas básicas | 1h |
| `02_estruturas_controle.ipynb` | if/elif/else, for, while, list comprehensions | 1h |
| `03_funcoes.ipynb` | Funções, parâmetros, retorno, lambda, *args/**kwargs | 1h |
| `exercicios/exercicio_01.ipynb` | Exercício prático: classificação de consumidores | 30min |

## Objetivos de Aprendizagem

Ao final deste módulo, você será capaz de:

- Declarar variáveis com os tipos corretos para dados do setor elétrico
- Usar estruturas de controle para implementar regras de negócio (tarifação, validação)
- Criar funções reutilizáveis para cálculo de indicadores (DEC, FEC)
- Manipular listas e dicionários para representar registros de consumidores

## Conceitos do Setor Abordados

- Unidade Consumidora (UC) e seus atributos cadastrais
- Modalidades tarifárias (A1, A2, A3, A4, AS, B1, B2, B3, B4)
- Classes de consumo (Residencial, Comercial, Industrial, Rural)
- Indicadores DEC e FEC (introdução ao cálculo)
- Estrutura de um registro de consumidor
