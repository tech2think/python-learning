Configuração do Ambiente com uv

Passo 1 — Instalar o uv

# Linux / Mac
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows (PowerShell)
powershell -ExecutionPolicy ByPass -c \
  "irm https://astral.sh/uv/install.ps1 | iex"
uv é o gerenciador moderno do ecossistema Python —
10–100x mais rápido que pip, sem necessidade de instalar Python separadamente.
