M1 Estruturas de Dados

Listas (Sequências)

Coleções ordenadas e mutáveis de itens.

# Criando uma lista de leituras
leituras = [310, 325, 342, 298, 315]

# Acessando o primeiro elemento (índice 0)
primeira_leitura = leituras[0]

# Adicionando um novo elemento
leituras.append(330)

Dicionários (Chave-Valor)

Coleções de pares com chaves únicas.

# Criando um registro de UC
consumidor = {
  "cod": 100042,
  "nome": "Maria Silva",
  "classe": "B1",
  "consumo_kwh": 342.7,
  "ativo": True
}

# Acessando valores
nome_cliente = consumidor["nome"]

Regras e Dicas

  • Listas: o primeiro item está sempre no índice 0.
  • Use len(lista) para saber o número de itens.
  • Listas preservam a ordem em que os itens foram inseridos.
  • Dicionários: as chaves (ex: "nome") devem ser únicas.
  • Acesso rápido: encontrar um valor pela chave é instantâneo.
  • Ideal para representar objetos complexos (clientes, medidores).
