M1 Estruturas de Controle

Classificação tarifária (if/elif)

def classificar_tarifa(consumo_kwh):
  if consumo_kwh <= 30:
    return "Baixa Renda"
  elif consumo_kwh <= 100:
    return "Faixa 1"
  elif consumo_kwh <= 220:
    return "Faixa 2"
  else:
    return "Faixa 3"

print(classificar_tarifa(342)) # → "Faixa 3"

Iteração sobre leituras (for)

leituras = [310, 325, 342, None, 315]

validas = []
for leitura in leituras:
  if leitura is not None:
    validas.append(leitura)

# List comprehension — mais pythonico
validas = [x for x in leituras
           if x is not None]

media = sum(validas) / len(validas)
print(f"Média: {media:.1f} kWh")

Regras e Dicas

  • Indentação é lei: Em Python, os blocos de código são definidos pelos espaços no início da linha (use 4 espaços).
  • Não esqueça: É obrigatório usar dois-pontos (:) no final das linhas de if, elif, else e for.
  • if/elif/else: Em uma cadeia de condições, apenas o bloco da primeira condição verdadeira será executado.
  • for loop: É a forma mais direta e pythônica de percorrer itens de uma lista um por um.
