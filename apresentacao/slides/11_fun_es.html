Cálculo do DEC

def calcular_dec(interrupcoes: list) -> float:
  """
  Calcula o DEC (Duração Equivalente de
  Interrupção por Consumidor).

  Args:
    interrupcoes: lista de dicts com
      'duracao_h' e 'consumidores'
  Returns:
    DEC em horas
  """
  total_ucs = interrupcoes[0]['total_ucs']
  soma = sum(
    i['duracao_h'] * i['consumidores']
    for i in interrupcoes
  )
  return round(soma / total_ucs, 4)

Validação de CPF

import re

def validar_cpf_formato(cpf: str) -> bool:
  """Valida formato do CPF."""
  cpf_limpo = re.sub(r'\D', '', cpf)
  return len(cpf_limpo) == 11

def formatar_cpf(cpf: str) -> str:
  """Formata CPF: 12345678900 →
  123.456.789-00"""
  c = re.sub(r'\D', '', cpf)
  if len(c) != 11:
    return cpf
  return f"{c[:3]}.{c[3:6]}.{c[6:9]}-{c[9:]}"

Regras e Dicas

  • Reutilização: Use funções para evitar repetir lógica de cálculo (como DEC/FEC) em vários lugares.
  • Docstrings: Sempre documente o que a função faz, seus parâmetros e o que ela retorna entre """triple quotes""".
  • Type Hints: Indicar tipos (ex: cpf: str) ajuda na leitura do código e evita bugs.
  • Modularidade: Mantenha funções focadas em uma única tarefa para facilitar testes e manutenção.
