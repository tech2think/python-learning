Lendo o cadastro de consumidores

import pandas as pd

df = pd.read_csv("dados/consumidores_simulado.csv")

# Exploração básica
print(df.shape) # (50, 12)
df.head() # primeiras 5 linhas
df.info() # tipos e nulos
df.describe() # estatísticas

# Verificar nulos
df.isnull().sum()

Filtros e agrupamentos

# Filtrar ativos
ativos = df[df['flg_ativo'] == True]

# Consumidores B1 com consumo > 300
altos = df[
  (df['cod_modalidade_tarifaria'] == 'B1')
  & (df['vlr_consumo_medio_kwh'] > 300)
]

# Agrupamento por classe
df.groupby('cod_classe_consumo')\
  ['vlr_consumo_medio_kwh']\
  .agg(['mean', 'count', 'sum'])\
  .round(2)

Regras e Dicas

  • DataFrame (df): É a estrutura principal do Pandas, funcionando como uma tabela onde cada coluna possui um tipo de dado definido.
  • Exploração: Sempre comece com df.info() para identificar colunas com valores nulos e tipos de dados errados.
  • Filtros: Use parênteses () para separar condições múltiplas e o símbolo & para o "E" lógico.
  • Agrupamentos: O groupby é o equivalente à Tabela Dinâmica, essencial para gerar indicadores por Classe ou UF.
