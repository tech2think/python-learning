Score de qualidade por registro

def score_qualidade(row) -> float:
  campos = [
    'nom_consumidor', 'cpf_cnpj',
    'cod_modalidade_tarifaria',
    'nom_municipio', 'num_cep',
    'num_medidor', 'dat_ultima_leitura'
  ]
  preenchidos = sum(
    1 for c in campos
    if pd.notna(row[c]) and row[c] != ''
  )
  return round(preenchidos / len(campos), 2)

df['score_qualidade'] = df.apply(
  score_qualidade, axis=1
)

Dimensões de qualidade

# Completude (%)
completude = (
  df['nom_consumidor'].notna().mean() * 100
)

# Unicidade (Duplicatas)
duplicados = (
  df.duplicated(subset=['cod_consumidor']).sum()
)

# Consistência
consistente = (
  df['cod_classe_consumo'].isin(['RES', 'IND', 'COM'])
)

Regras e Dicas

  • Completude: Verifique se campos críticos (CPF, Medidor) estão preenchidos para evitar erros de faturamento.
  • Unicidade: O cod_consumidor deve ser único para garantir a integridade da base.
  • Consistência: Cruze dados (ex: Classe vs. Tarifa) para validar se as regras de negócio foram aplicadas.
  • Score: Use o .apply(axis=1) para calcular indicadores linha a linha de forma simples.
