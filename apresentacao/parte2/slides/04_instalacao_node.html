M4 Pré-requisito: Instalando o Node.js

Tanto o Claude Code quanto o Gemini CLI são ferramentas desenvolvidas em JavaScript/TypeScript e distribuídas como pacotes do Node.js (via npm).

Por que precisamos disso?

  • O Node.js é o motor que executa o código dessas ferramentas no seu computador, fora do navegador.
  • O NPM (Node Package Manager) é o instalador automático, que baixa a IA e todas as suas dependências.

Passo a Passo

1. Windows / macOS
Acesse nodejs.org e baixe a versão LTS (Long Term Support). Siga o instalador padrão (Next, Next, Finish).
2. Linux (Debian/Ubuntu)
# Instale o NVM (Node Version Manager)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# Instale a versão mais recente LTS
nvm install --lts
3. Verificar Instalação
node -v # Exemplo: v20.11.0
npm -v # Exemplo: 10.2.4
