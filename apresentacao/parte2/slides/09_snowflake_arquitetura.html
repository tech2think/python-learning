M5 Snowflake + Python: Arquitetura

Componentes Snowflake

AccountInstância da sua empresa
WarehouseCluster de computação (VXS a 6XL)
DatabaseBanco de dados lógico
SchemaAgrupamento de tabelas/views
Snowflake cobra por tempo de execução do warehouse. Sempre suspenda ao terminar análises longas.

Conexão Python

import snowflake.connector
from dotenv import load_dotenv
import os

load_dotenv()

conn = snowflake.connector.connect(
  account=os.getenv("SF_ACCOUNT"),
  user=os.getenv("SF_USER"),
  password=os.getenv("SF_PASSWORD"),
  warehouse=os.getenv("SF_WAREHOUSE"),
  database=os.getenv("SF_DATABASE"),
  schema=os.getenv("SF_SCHEMA"),
)

cursor = conn.cursor()
cursor.execute("SELECT CURRENT_VERSION()")
print(cursor.fetchone()[0])
