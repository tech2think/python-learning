M4 Markdown para Documentação Técnica

Por que Markdown?

  • Legível como texto puro
  • Renderizado em GitHub, Jupyter, VS Code
  • Base para dicionários de dados
  • Gerado automaticamente pelo Claude Code
  • Versiona junto com o código (git)
O Claude Code lê e gera Markdown. Use-o para criar e manter documentação técnica automaticamente.
