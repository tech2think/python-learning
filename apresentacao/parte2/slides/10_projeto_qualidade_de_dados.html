M5 Projeto: Qualidade de Dados de Consumidores

Objetivo: Construir um pipeline Python que conecta ao Snowflake, analisa a qualidade do cadastro de UCs (Unidades Consumidoras), gera um relatório com score por registro e identifica os principais problemas para correção.

1. Extração

  • Conectar ao Snowflake
  • Carregar tabela CONSUMIDORES_UC
  • Filtros por data de competência

2. Análise de Qualidade

  • Completude por campo
  • Validação CPF/CNPJ
  • Consistência tarifária
  • Outliers de consumo

3. Relatório

  • Score por UC (0 a 1)
  • Dashboard Plotly
  • Exportar Excel com issues
  • Envio de alertas
