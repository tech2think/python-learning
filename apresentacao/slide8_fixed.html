M1 Variáveis e Tipos de Dados

Tipos básicos no setor elétrico

# Inteiros — códigos e tensões
cod_consumidor = 100042
tensao_nominal = 220 # Volts

# Float — medições
consumo_kwh = 342.7
fator_potencia = 0.92

# String — cadastro
nome = "Maria Silva Santos"
municipio = "São Paulo"

# Booleano — status
ativo = True
inadimplente = False

Estruturas de dados

# Lista — leituras mensais
leituras = [310, 325, 342, 298, 315]

# Dicionário — registro UC
consumidor = {
  "cod": 100042,
  "nome": "Maria Silva",
  "classe": "B1",
  "consumo_kwh": 342.7,
  "ativo": True
}

# Acessando valores
print(consumidor["nome"])

Regras Importantes

  • Tipagem dinâmica: não precisa especificar o tipo (int, str, etc.) na criação.
  • Nomes não podem ter espaços ou caracteres especiais (exceto _).
  • Case Sensitive: a variável X é diferente da variável x.
  • Proibido usar palavras reservadas (if, def, True, etc.) como nomes.
  • Use # para inserir comentários no seu código.
