# Módulo 4 — Claude Code & IA no Desenvolvimento

Aprenda a usar Claude Code e técnicas de prompt engineering para acelerar o desenvolvimento de análises de dados no setor elétrico.

## Notebooks

| Arquivo | Conteúdo | Duração |
|---------|----------|---------|
| `01_introducao_ia_desenvolvimento.ipynb` | O que é Claude Code, casos de uso, fluxo de trabalho | 1h |
| `02_prompt_engineering.ipynb` | Princípios de prompts eficazes, exemplos para dados | 1h |
| `03_estrutura_projetos_claude.ipynb` | CLAUDE.md, skills, organização de projetos | 1h |

## Arquivos de Exemplo

| Arquivo | Descrição |
|---------|-----------|
| `exemplos/CLAUDE.md` | Exemplo de CLAUDE.md para projeto de qualidade de dados |
| `exemplos/exemplo_skill.md` | Skill de verificação de qualidade de dados |
| `exemplos/exemplo_prompt.md` | Template de prompt para análise de cadastro |

## Pré-requisitos

- Node.js 18+ instalado (`node --version`)
- Conta e API key da Anthropic (anthropic.com)
- Claude Code instalado: `npm install -g @anthropic-ai/claude-code`
