# Módulo 3 — Visualização de Indicadores

Aprenda a criar gráficos profissionais para o setor elétrico usando matplotlib, seaborn e plotly.

## Notebooks

| Arquivo | Conteúdo | Duração |
|---------|----------|---------|
| `01_visualizacao_indicadores.ipynb` | Gráficos de barras, heatmaps, gráficos interativos, DEC/FEC | 2h |

## Bibliotecas

| Biblioteca | Uso |
|------------|-----|
| `matplotlib` | Gráficos estáticos, exportação para PDF/PNG |
| `seaborn` | Gráficos estatísticos, heatmaps de qualidade |
| `plotly` | Gráficos interativos para dashboards |

## Dicas

- Use `matplotlib` quando precisar exportar gráficos para relatórios em Word/PDF
- Use `plotly` quando precisar de gráficos interativos no Jupyter ou em dashboards web
- Use `seaborn` para análises estatísticas e heatmaps de correlação/qualidade
